Global Mappings

Bellow is a list of all default global key mappings provided by Vimwiki. By default, <Leader> equals to "\". 

<Leader>ww		Open index file.
<Leader>wt		Open index file on new tab.
<Leader>ws		List and select available wikis.
<Leader>wi		Open diary index file.
<Leader>w<Leader>w	Open diary wiki-file for today.
<Leader>w<Leader>t	Open diary wiki-file for today in new tab.
<Leader>w<Leader>y	Open diary wiki-file for yesterday.
<Leader>w<Leader>m	Open diary wiki-file for tomorrow.

Local Mappings

Bellow is a list of all local key mappings provided by Vimiki. As local settings, they're available when FileType is set to "vimwiki", that is, when you are working on a vimwiki file.

Glossary: 

<Leader> 	"\" key
<CR> 		"Enter" key
<S>		"Shift" key
<C>		"Control" key
<A>		"Alt" key

Mappings on Insert mode: 

MAP			DESCRIPTION

<CR>			In a list item, insert a new bulletd/number in the next
			line, numbers are incremented. In an empty list item,
			delete the item symbol. This is useful to end a list,
			simply press <CR> twice.
<S-CR>			Does not insert a new list item, useful to create
			multilined list items.
<C-T>			Increase the level of a list item.
<C-D>			Decrease the level of a list item.
<C-L><C-K>		Change the symbol of the current list item to the prev
			available. From - to a) to I) to * to 1.
<C-L><C-M>		Create/remove a symbol from a list item.

Text objects

MAP	DESCRIPTION

ah	A header including its content up to the next header. 
ih	The content under a header (like 'ah', but excluding the header itself and 
	trailing lines).
aH	A header including all of its subheaders. When [count] is 2, include the
	parent header, when count is 3, the grandparents and so on.
iH	Like 'aH', but excluding the header itself and trailing empty lines.
